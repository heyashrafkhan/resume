# Hey Ashraf Khan — Resume Website

This repository contains the source files for my personal resume website.

## Overview

The site is a lightweight, responsive static website that presents my professional summary, skills, experience, education, and contact information.

## View locally

1. Clone the repository:

   git clone https://github.com/heyashrafkhan/resume.git
2. Open `index.html` in your browser, or serve the folder locally:

   - Using Python 3:
     python -m http.server 8000
     Visit http://localhost:8000

## Deployment

You can host this site on GitHub Pages (or any static hosting provider). To use GitHub Pages:

1. Push the repository to GitHub (if not already pushed).
2. In the repository Settings -> Pages, set the source to the branch and folder where `index.html` lives (usually `main` / `/` or the `gh-pages` branch).

## Project structure

- index.html — Main page of the resume site
- assets/ — Images, icons, fonts
- css/ — Stylesheets
- js/ — Optional JavaScript

Modify the HTML/CSS files to update content and styling.

## Customize

- Replace text in `index.html` with your own name, summary, experience, and education.
- Update images and icons in `assets/`.
- Edit styles in `css/` to change colors, fonts, and layout.
